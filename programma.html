NMCT programma

NMCT is een praktijkgerichte professionele IT bachelor opleiding rond nieuwe technologieën aan Howest. In de eerste 3 semesters bouw je een stevige, allround IT basis op om daarna te kiezen uit 1 van onze 3 uitstroomprofielen: Web & App Developer, Smart Technology Creator of Infrastructure Engineer.

Smart
Technology
Creator

Wil je sensordata verzamelen en met elkaar connecteren? Is het ontwerpen en programmeren van een artificieel brein echt iets voor jou? Lijkt het je wel iets om je voorspellingen visueel weer te geven in slimme, mixed reality applicaties? Dan is NMCT uitstroomprofiel Smart Technology Creator waarschijnlijk iets voor jou! Programmeer en connecteer sensoren met Lora en Python. Analyseer datastromen met Machine Learning algoritmes en neurale netwerken.

Jouw toekomst

Big Data Architect, Cloud Engineer, R&D Developer, IT Engineer, SQL Programmeur, …

Web
& App
Developer

Wil je zowel de front-end als back-end van slimme applicaties bouwen? Is verhalen visueel weergeven echt iets voor jou? Kriebelt het om één van de eerste echte Mixed Reality Developers te worden? Dan is NMCT uitstroom profiel Web & App Developer waarschijnlijk iets voor jou! Ga aan de slag met HTML, Javascript en Xamarin. Voel je helemaal thuis in Android en Java. Frameworks en patterns hebben geen geheimen voor jou!

Jouw toekomst

Big Data Architect, Cloud Engineer, R&D Developer, IT Engineer, SQL Programmeur, …

Infrastructure
Engineer

Word je enthousiast van computernetwerken installeren, configureren en onderhouden? Is een computernetwerk beveiligen tegen externe en interne bedreigingen echt iets voor jou? Dan is NMCT uitstroomprofiel Infrastructure Engineer waarschijnlijk iets voor jou! Leer alles over infrastructuur opzetten met behulp van Windows en Linux, zowel lokaal als in de Cloud. Wordt expert in VM Ware een virtueel computernetwerk automatisch deployen aan de hand van scripting.

Jouw toekomst

Big Data Architect, Cloud Engineer, R&D Developer, IT Engineer, SQL Programmeur, …
